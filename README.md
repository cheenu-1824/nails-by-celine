# Nails By Celine

A full-stack web application built using the **MERN stack (MongoDB, Express, React, Node.js)** for a nail salon business.  
The website allows customers to explore nail services, view designs, and interact with the business online.

---

## Tech Stack

**Frontend**
- React
- JavaScript (ES6+)
- HTML5
- CSS3
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB (Mongoose)

**Other Tools**
- Git & GitHub
- npm
- REST APIs

---

## Features

- Responsive modern UI
- Service listings (nail designs, pricing, etc.)
- Client-side routing with React
- Backend API built with Express
- MongoDB database integration
- Scalable architecture for future features such as:
  - Online bookings
  - Admin dashboard
  - User authentication
  - Payment integration

---

## Getting Started

### Prerequisites
- Node.js
- npm
- MongoDB (local or cloud)

### Installation

```bash
git clone https://github.com/cheenu-1824/nails-by-celine.git
cd nails-by-celine

cd server
npm install

cd ../client
npm install

### Running the Program

cd server
npm start

cd client
npm start
